ALAK: Deterministic Audio Packaging (Beta)

Two reversible, streamable pipelines for WAV/PCM → compact containers (.alak / .alak.bin).

What it is

ALAK provides two time-domain pipelines—Standard (Δ→zigzag→RLE) and Base-i (sparse index/value)—with tunable decimal precision (dp1–dp4) and compact JSON or MsgPack containers. Round-trip is exact for integer-quantized inputs at the chosen precision; on speech/music, typical settings yield ≈55–83 dB SNR with 1.5×–4.0× size reduction vs WAV (content-dependent).

Download (Windows 10+ / macOS 12+)

Windows: you may see SmartScreen for unsigned binaries → More info → Run anyway.

Key results (current beta)

  • Size vs WAV: ~1.5×–4.0× smaller at dp3–dp4; external .bz2/.zst can add 20–60% shrink.
  • Quality: speech/music ≈55–83 dB SNR; structured signals reconstruct with MSE ≈ 0 (within numeric precision).
  • Deterministic, streamable containers; CLI/GUI for reproducible testing.

Research: Mod-N (exploratory)

Mod-N generalizes Base-i to cyclic phase sets (incl. irrationals). In small-qubit IBM tests (e.g., Torino), ensembles with phase staggering (π, e, φ) observed ~5–15% higher state-fidelity vs matched QFT baselines. Not part of desktop builds; methods/data available under NDA.
